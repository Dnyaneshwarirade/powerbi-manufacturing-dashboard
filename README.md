# Manufacturing Performance Dashboard

## 📌 Overview

This project is an interactive Power BI dashboard developed to analyze manufacturing performance across different business areas.

The dashboard provides actionable insights into production efficiency, product quality, inventory management, financial performance, and sustainability metrics.

---

## 🚀 Features

- Executive KPI Dashboard
- Production Analysis
- Quality Control Dashboard
- Inventory Management
- Financial Performance Analysis
- Sustainability Monitoring
- Interactive Filters & Drill-downs
- Dynamic Visualizations

---

## 📊 KPIs Included

- Total Production
- Revenue
- Profit
- Production Cost
- Inventory Value
- Defect Rate
- Waste Generated
- Sustainability Score
- Customer Satisfaction
- Machine Utilization
